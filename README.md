# 俄罗斯方块游戏

一个经典的俄罗斯方块益智游戏，使用HTML5、CSS3和JavaScript实现。

## 游戏特性

### 🎮 核心玩法
- **7种方块**: I、O、T、S、Z、J、L 型方块，每种都有独特的颜色
- **流畅控制**: 键盘控制方块移动、旋转和下落
- **智能消除**: 填满一行自动消除得分
- **难度递增**: 每消除10行升级，游戏速度加快
- **实时统计**: 分数、等级、消除行数、游戏时间

### 🎯 操作说明
- **← → 键**: 左右移动方块
- **↓ 键**: 加速下落（获得额外分数）
- **↑ 键 / Z 键**: 旋转方块
- **空格键**: 直接到底（硬降）
- **P 键**: 暂停/继续游戏

### 🎨 界面特色
- **现代化设计**: 深色主题，渐变色彩
- **响应式布局**: 支持不同屏幕尺寸
- **动画效果**: 方块发光、平滑过渡
- **预览功能**: 显示下一个方块

## 如何运行

1. 直接在浏览器中打开 `tetris.html` 文件
2. 游戏会自动开始，方块从顶部落下
3. 使用键盘控制方块
4. 当方块堆积到顶部时游戏结束

## 技术实现

- **HTML5**: 语义化结构
- **CSS3**: Grid布局、动画效果、响应式设计
- **JavaScript**: ES6+ 语法，模块化设计
- **Canvas替代**: 使用CSS Grid实现游戏板

## 分数系统

只有通过**消除行**才能获得分数：

- **单行消除**: 40 × 等级
- **双行消除**: 100 × 等级
- **三行消除**: 300 × 等级
- **四行消除** (Tetris): 1200 × 等级

游戏的目标是通过战略性摆放方块来消除完整的行，获得高分！
